# AGAINST_THE_MUSIC_v_0.10
Игра Against the Music представляет из себя что-то наподобие буллет хелла, с бесконечным геймплеем и счетом


Контроли!

В меню: 
 W/S - перемещение по блокам
 J - выбор
В игре:
 WASD - движение
 K/N/M/, - стрельба по направлению
 J - дэш
 L - пустышка
